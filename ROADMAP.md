# Roadmap

## Current Status

The core API is live and functional. Real-time wallet behavioral analysis is working via Helius data integration, returning archetype classification, risk scoring, behavioral patterns, and portfolio overview from any Solana wallet address.

---

## Phase 1 — Core API (Current)

**Goal:** Functional behavioral analysis API with real on-chain data.

- [x] REST API with wallet analysis endpoint
- [x] Real-time data via Helius API integration
- [x] Wallet archetype classification (6 types)
- [x] Behavioral pattern detection
- [x] Risk scoring engine (0–100)
- [x] Portfolio overview
- [x] Input validation and error handling
- [x] JavaScript SDK (basic)
- [x] API documentation

---

## Phase 2 — MVP Completion *(Grant Funded)*

**Goal:** A publicly accessible, usable product that developers can integrate immediately.

**Timeline:** Q2 2026

- [ ] Public demo interface — connect a wallet and see your behavioral profile
- [ ] Expanded pattern library (10+ behavioral signals)
- [ ] Confidence scoring improvements based on transaction volume
- [ ] Historical trend analysis (30/90 day behavioral shifts)
- [ ] Rate limiting and API key management
- [ ] Hosted public endpoint (no self-hosting required)
- [ ] Developer onboarding documentation



---

## Phase 3 — Ecosystem Integration

**Goal:** Make Strata BI a composable data layer that other Solana projects can build on.

**Timeline:** Q3 2026

- [ ] Multi-wallet batch analysis endpoint
- [ ] Webhook support (get notified when a wallet's behavior changes)
- [ ] Bot detection signals
- [ ] Anomaly detection for unusual transaction patterns
- [ ] Integration guides for DeFi protocols, NFT platforms, and wallets
- [ ] SDK expansion (Python, Rust)

---

## Phase 4 — Intelligence Layer

**Goal:** Move from reactive analysis to predictive behavioral modeling.

**Timeline:** Q4 2026

- [ ] Behavioral clustering (identify wallet groups with similar patterns)
- [ ] Risk signal API for security and compliance use cases
- [ ] On-chain reputation scoring
- [ ] AI-generated behavioral summaries with deeper context
- [ ] Public dataset for Solana ecosystem researchers

## Long-Term Vision

Strata BI aims to become the default behavioral intelligence layer for the Solana ecosystem — the API that developers reach for when they need to understand what a wallet actually does, not just what transactions it has made.
