// overview
Strata BI provides real-time behavioral intelligence for Solana wallets. Pass any wallet address and receive archetype classification, risk scoring, portfolio overview, and behavioral pattern detection — all in a single API call.
// base url
https://solana-behavioral-intelligence-api-production.up.railway.app
// endpoint
GET /api/v1/solana/wallet/{address}/analysis
Analyze a Solana wallet address and return behavioral intelligence data.
// path parameters
parameter type description
address string Base58-encoded Solana wallet address (32–44 characters). Mainnet only.
// example request
curl
curl "https://solana-behavioral-intelligence-api-production.up.railway.app/api/v1/solana/wallet/YOUR_WALLET_ADDRESS/analysis"
// response fields
field type description
